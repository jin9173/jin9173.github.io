while 반복문 (loop)

조건부분이 True일 동안에 수행부분이 계속 실행됨

  1. 파이썬은 조건부분의 결과값(불린값)이 True인지 False인지 체크한다.
  2. 조건 부분이 True면 파이썬은 반복문의 수행부분을 실행한다. 그리고 수행부분이 끝나면 다시 조건 부분으로 돌아간다.
  3. 조건 부분이 False면 파이썬은 while 반복문의 수행부분을 건너 뛰어 그 이후의 코드를 실행한다.

if 조건문 (conditional)

조건부분이 참일 때 수행부분이 딱 한번만 실행됨

  1. 처음에 if문은 꼭 써야한다.
  2. 중간에 elif문은 0번이상, 원하는 횟수만큼 쓸 수 있다.
  3. 마지막에 else문은 0번 또는 1번 쓸 수 있다.

if문의 조건부분이 한번 실행된다. 결과적으로 a는 b보다 작습니다.가 출력된다. 만약 a가 b보다 크거나 같았다면 if문은 실행되지 않고 아무것도 출력되지 않을 것이다.


break 문

while문의 조건부분과 상관없이 반복문에서 나오고 싶을 때 사용

continue문

현재 진행되고 있는 수행부분을 중단시키고 바로 조건부분을 확인하고 싶을 때 사용
