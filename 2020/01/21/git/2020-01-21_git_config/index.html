Git config 설정 해제 후 재등록

2020-01-21 14:01:50
Git

문제

커밋과 푸시를 제대로 했는데 잔디가 채워지지 않았다.


해결

git config 에 정확한 정보가 있는지 확인한다.

이메일 정보가 틀린 것을 확인했다. 잘못된 내용을 삭제한다. (나는 global 로 사용자 설정을 해놓았기 때문에 global 로 지워야 한다.)

사용자 설정 정보가 제대로 지워졌는지 확인한다.

잘 지워진 것을 확인했으니 git config 를 다시 설정한다. (편하게 global 로 설정)

사용자 정보가 잘 추가됬는지 확인한다.

커밋과 푸시를 해보니 잔디가 잘 채워지는 것을 확인했다.

문제 해결!
