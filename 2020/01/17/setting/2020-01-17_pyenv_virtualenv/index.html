가상환경

: 프로젝트별로 독립된 버전의 가상환경을 만들어주는 python에서 제공하는 툴

인터넷에서 다운로드 된 파이썬 라이브러리들이 충돌을 일으키는 것을 방지하기 위함

  • virtualenv: 가상환경의 인터프리터 시스템의 디폴트 파이썬으로 잡아주기 때문에, CentOS인 경우 Python 2.7로 설정된다. (물론 --python 옵션을 주면 Python 3.X 버전도 선택 가능)
  • venv: 위와 같은 옵션을 신경쓸 필요없이 명령줄에 사용된 파이썬을 그대로 인터프리터로 설정한다.
