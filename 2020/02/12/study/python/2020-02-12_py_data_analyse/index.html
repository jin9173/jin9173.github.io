파일 읽기

첫번째 파라미터로 파일이름을 넘겨주고, 두번째 파라미터로 파일을 읽는다는 의미에서 read의 약자인 r을 넘겨준다.

파일을 쓸 때는 write의 약자인 문자열 w를 넘겨준다.


strip()

: 가장 앞쪽과 뒤쪽에 있는 화이트 스페이스가 제거된 새로운 문자열을 리턴함


split()

: 파라미터로 넘겨주는 값을 기준으로 문자열을 분리시키고, 분리된 값들이 정리되어 있는 리스트를 리턴해준다.

파라미터가 없는 경우

: 아무 값도 넘겨주지 않았을 경우 기본값으로 ""가 넘어가기 된다. 즉, 공백(화이트 스페이스)을 기준으로 문자열을 나눈다.
