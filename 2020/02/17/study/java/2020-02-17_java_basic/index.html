변수(Variable)

: 정보를 저장하고 쓸 수 있게 해주는 “이름표”같은 개념

자료형(Data Type)

: 자료들의 종류 (문자열, 숫자형, 논리형..)

함수(Function)

: 변수가 값을 보관하는 역할을 한다면, 함수(function)는 명령들을 보관하는 역할을 한다. 동일한 내용을 반복하여 쓰고 있거나, 프로그램을 깔끔하게 정리하고 싶을 때 함수를 사용한다.

수학에서는 x라는 입력값을 넣었을 때, y라는 결과값이 나오는 것을 함수라고한다. 프로그래밍에서의 함수도 비슷하다. 입력값을 넣은 후, 그 값을 사용하여 필요한 동작을하고, 출력값을 만들어낸다. 이 때, 함수의 입력값을 “파라미터” 혹은 “매개변수”라고 부르며, 출력값을 “리턴값”이라고 부른다.

함수는 대부분 () 와 {} 를 사용한다.

함수와 자료형

함수의 입력은 “파라미터”, 출력은 “리턴값”이라고 했다.

자바에서는 함수의 입력은 어떤 자료형인지, 출력은 어떤 자료형인지 미리 정해준다. 그리고 맞지 않는 자료형을 입력하거나 출력하게 되면 오류가 발생한다.

배열(Array)

: 여러 개의 값을 하나의 변수에 담고 싶을 때 배열을 사용한다.

배열은 주로 [] 를 사용하여 표현한다.
