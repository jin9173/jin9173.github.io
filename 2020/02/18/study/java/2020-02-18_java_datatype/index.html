기본 자료형 (Primitive Types)

: 가장 기본이되는 자료형

Type Bits Range of Values
byte 8bits -2^7 ~ 2^7-1 (-128 ~ 127)
short 16bits -2^15 ~ 2^15-1 (-32768 ~ 32767)
int 32bits -2^31 ~ 2^31-1 (-2147483648 ~ 2147483647)
long 64bits -2^63 ~ 2^63-1 (-9223372036854775808 ~ 9223372036854775807)
float 32bits *single-precision 32-bit IEEE 754 floating point
double 64bits *double-precision 64-bit IEEE 754 floating point
char 16bits \u0000 ~ \uffff (0 ~ 2^15-1)
boolean *VMD true, false

각 자료형의 기본값

type 기본 값
byte 0
short 0
int 0
long 0L
float 0.0f
double 0.0d
char ‘\u0000’
boolean false

1. 숫자형 (Number)

: 숫자를 담는 자료형으로써 정수형(byte, short, int, long)과 소수형(float, double)이 있다.


2. 글자 (Character)

: 글자 하나를 담는 자료형 char은 딱 글자 하나만 넣어줄 수 있고, 작은 따옴표로 글자를 둘러싸야 한다. 글자 여러개를 담고 싶으면 큰 따옴표로 둘러싸고 아래에서 배울 String이라는 자료형을 사용한다.

4. 문자열 (String)

: String은 기본자료형이 아니라 클래스이다. 클래스를 변수의 형으로 쓰고 변수를 선언하면 그 변수는 클래스의 인스턴스를 담을 수 있다.
